# Project Overview — Brochure as a Service

Brochure as a Service (BaaS) is a two-part system:

1. **Engine** — Generates single-page brochure websites for small South African businesses from JSON data using Handlebars templates and Claude API
2. **Pipeline** — Lead prospector that searches Google Maps for SA businesses without websites and logs them into a tracker spreadsheet

## Business Context

Target market: Small South African service businesses (plumbers, salons, restaurants, etc.) that have a Google Maps listing but no website. The system generates affordable, professional brochure sites and the pipeline finds qualified leads.

## Key Stakeholder

- **Agile Bridge** (agilebridge.co.za) — the brand behind the generated websites. All footer credits must read "Website by Agile Bridge".
